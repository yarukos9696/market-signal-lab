# Model Card

## Intended use

Private research and paper/live-style evaluation of short-horizon crypto-perpetual market signals.

## Public release status

Model binaries, exact features, and training code are not published. This public repo contains aggregate metrics and documentation only.

## Evaluation

Public aggregate selected-baseline result:

- 782 trades;
- $2,653.36 net PnL;
- 40.15% win rate;
- $277.85 max drawdown;
- PnL/DD 9.55;
- 3 negative traded windows.

## Risks

- regime shift;
- overfitting;
- execution/slippage sensitivity;
- data-quality failures;
- live/backtest drift.

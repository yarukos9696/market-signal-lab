# Data Sheet

## Data types

The private system uses crypto-perpetual market data such as OHLCV, open interest, funding, and derived market-state context.

## Public data release

No raw data is published. Only aggregate metrics are included.

## Reason for withholding raw data

- size and licensing/collection constraints;
- operational sensitivity;
- potential strategy leakage through timestamps and symbols;
- privacy/security concerns around live runtime logs.

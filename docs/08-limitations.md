# Limitations

- Historical WFO/OOS replay is not a guarantee of future performance.
- Crypto regimes shift quickly; edge can decay.
- Execution quality, fees, slippage, latency, and outages matter.
- Public results are aggregate-level; raw trade logs are private.
- Private feature formulas and production implementation are intentionally withheld.
- A low number of trades in some windows can be either desirable filtering or insufficient opportunity coverage; it requires ongoing monitoring.

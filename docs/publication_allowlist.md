# Publication Allowlist

## Allowed in this public repo

- aggregate WFO/OOS metrics;
- aggregate monthly/window/side summaries;
- public charts generated from aggregate CSVs;
- architecture diagrams;
- alpha thesis and methodology notes;
- limitations and lessons learned.

## Not allowed

- API keys, tokens, `.env`, credentials;
- account IDs, order IDs, wallet/exchange identifiers;
- raw historical market data, DBs, parquet files;
- raw live/paper logs or trade-by-trade ledgers;
- exact private feature formulas;
- model binaries, calibrators, private configs;
- full production execution adapters;
- private git history.

## Principle

Publish enough to demonstrate real research and engineering discipline, but not enough to reconstruct the private trading system.
